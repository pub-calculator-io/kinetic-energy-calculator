== Description ==

Kinetic energy calculator easily finds kinetic energy, mass or velocity with the kinetic energy formula KE = 1/2 mv². Easy to use KE calculator.

Libraries in use:
1. https://mathjs.org/
2. https://katex.org/
3. https://github.com/aFarkas/lazysizes
4. https://github.com/RobinHerbots/Inputmask
5. https://air-datepicker.com/
6. https://www.chartjs.org/

== Installation ==

1. Upload the /ci_kinetic_energy_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the [Kinetic Energy Calculator](https://www.calculator.io/kinetic-energy-calculator/ "Kinetic Energy Calculator Homepage") plugin through the "Plugins" menu in WordPress.
3. Configure your settings via Appearance > Widgets > Kinetic Energy Calculator.
4. And you're good to go!
